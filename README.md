This repository contains a complete PyTorch implementation of an end-to-end deep learning pipeline designed to reconstruct high-resolution 3D brain surfaces (specifically the Left Hemisphere White Matter) directly from raw 3D MRI volumes. 

It leverages a hybrid architecture combining a **3D Convolutional Vision Transformer (CvT-24)** for volumetric feature extraction and a **Window-Based Graph Attention Network** to deform a spherical template into complex cortical folds.

## 🏗️ Architecture Overview

The model is split into two massive components that communicate to turn pixels into vertices:

### 1. The Volumetric Encoder (3D CvT)
Inspired by Convolutional Vision Transformers (CvT), the encoder processes the `[1, 96, 144, 192]` MRI volume.
* **Hierarchical Tokenization:** Uses 3D convolutions to gradually reduce spatial dimensions while increasing channel depth, preserving local anatomical structures.
* **3D Attention:** Applies multi-head self-attention across the 3D volume to understand global brain context.

### 2. The Mesh Graph Decoder
Instead of predicting voxels, the decoder predicts coordinates, deforming a starting spherical template (Icosphere) to match the target brain.
* **Geodesic Window Partitioning:** Groups the 40,962 vertices of the template into localized windows based on their graph connectivity.
* **Window-Based Graph Attention:** Vertices pay attention to their neighbors to ensure smooth, realistic folding without mesh self-intersection.

### 3. Loss Formulation
The model is optimized using a custom `BrainRegistrationLoss`:
* **Chamfer Distance:** Ensures the predicted 3D points physically overlap with the Ground Truth surface.
* **Edge Strain Loss:** Acts as a physical regularizer, preventing the triangles in the mesh from stretching too far or tearing.

---

## 🎯 Key Feature: Template Pre-Alignment
To drastically reduce training time, this repository includes a **Coordinate Pre-Alignment module**. 
Instead of forcing the neural network to learn how to translate and scale a generic sphere across the scanner space, the initial template is mathematically shifted to the Exact Origin and Scaled (via Standard Deviation) to match the real brain *before* the forward pass. 

This allows the network to dedicate 100% of its parameters to learning complex cortical folding from Step 1.
